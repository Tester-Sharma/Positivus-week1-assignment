PROJECT ANALYSIS: POSITIVUS
===========================

OVERVIEW
--------
Positivus is a responsive single-page landing page for a digital marketing agency.
Built with: Vanilla JavaScript, Vite, and Tailwind CSS.
Architecture: Component-based using native ES Modules.

FILE STRUCTURE & ANALYSIS
=========================

ROOT DIRECTORY
--------------
1. index.html
   - Main entry point.
   - Links to Google Fonts (Space Grotesk).
   - Contains the <div id="app"> container.
   - Loads /src/main.js.

2. package.json
   - Defines project metadata and dependencies.
   - Scripts:
     * dev: Starts local server (Vite).
     * build: Compiles for production.
     * preview: Previews the build.
   - Dependencies: tailwindcss, postcss, autoprefixer, vite.

3. postcss.config.js
   - Configures PostCSS with Tailwind and Autoprefixer plugins.

4. tailwind.config.js
   - Configures Tailwind CSS.
   - Custom Colors: positivus-green (#B9FF66), positivus-dark (#191A23), positivus-grey (#F3F3F3).
   - Custom Font: Space Grotesk.

SOURCE DIRECTORY (src/)
-----------------------
1. src/main.js
   - Application Entry Point.
   - Imports styles and component setup functions.
   - initApp(): Sets up the main HTML structure (header, sections, footer) and initializes all components.

2. src/style.css
   - Imports Tailwind directives (base, components, utilities).

3. src/constants.js
   - Centralized data storage.
   - Contains: NAV_LINKS, SERVICES_DATA, COMPANY_LOGOS, CONTACT_INFO, SOCIAL_LINKS.

4. src/counter.js
   - Unused utility function (likely from Vite template).

COMPONENTS (src/components/)
----------------------------
1. src/components/Header.js
   - Renders the navigation bar.
   - Features: Logo, Desktop Links, "Request a quote" button, Mobile Menu Icon.

2. src/components/Hero.js
   - Renders the top hero section.
   - Features: Headline, CTA button, Illustration, and Company Logos grid.

3. src/components/Services.js
   - Renders the Services section.
   - Generates a grid of cards based on SERVICES_DATA.
   - Cards have dynamic styling (green, dark, grey backgrounds).

4. src/components/CTA.js
   - Renders a "Call to Action" banner.
   - Features: "Let's make things happen" text, proposal button, and abstract background shapes.

5. src/components/Footer.js
   - Renders the footer.
   - Sections: Navigation, Social Links, Contact Info, Newsletter Form, Copyright.

6. src/components/Icons.js
   - SVG Library.
   - Exports functions returning SVG strings (LogoIcon, MenuIcon, HeroIllustration, etc.).
   - Allows customization via className arguments.
